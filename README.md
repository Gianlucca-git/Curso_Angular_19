# **_Angular 19_**

## Comandos

- instalar Angular version 19

> npm install -g @angular/cli@19

- iniciar proyecto

> ng new [NOMBRE_PROYECTO]
> css
> SSR (Server-Side Rendering) / N

- abrir servicio

> ng serve -o

- crear componente

> ng g c [ROOT/NAME]

- Contruir APP productiva

> ng build
